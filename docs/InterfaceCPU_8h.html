)

Generate an isothermal-beta model, from an azimuth and elevation array.

Returns an array of shape azimuth * elevation, containing the optical depth for each pointing.

Parameters
AzArray containing azimuth points in arcsec.
ElArray containing elevation points in arcsec.
n_AzNumber of azimuth points.
n_ElNumber of elevation points.
ibetaBeta parameter of isothermal model.
ne0Central electron number density, in electrons / cm**3.
thetacCore radius of cluster in arcsec.
DaAngular diameter distance in Megaparsec.
outputArray for storing outputs.
gridWhether or not to evaluate on Az-El grid, or along Az-El trace. Note: if grid=false, n_Az must equal n_El, and output must equal either one. If grid=true, n_Az does not need to equal n_El, output should have size n_Az*n_El.
