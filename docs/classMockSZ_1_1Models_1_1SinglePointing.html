Detailed Description

Class for generating a single pointing SZ signal.

Can choose between tSZ, kSZ and ntSZ (powerlaw).

Generate a single pointing signal of the kSZ effect.

Note that this method, in contrast to the other methods in this class, only returns the CMB distortion. The CMB can be added by adding the kSZ signal to a tSZ or ntSZ signal containing the CMB.

Parameters
nu_arrNumpy array of frequencies for kSZ effect, in Hz.
v_pecPeculiar velocity.
tau_eOptical depth along sightline.
n_muNumber of scattering direction cosines to include.
Returns
res 1D array containing kSZ effect.
